How to scroll an EditText component

Use Case: Read something from a datasource (from a file or a database) and display all the contents inside a non-editable view (EditText).

Solution: Use an EditText view to display the non-editable text. It will not scroll out of the box, you will need another view to do the scrolling. Wrap the EditText inside a ScrollView. The ScrollView will act as a viewport to textual content of the EditText.

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:orientation="vertical"
    android:layout_width="fill_parent"
    android:layout_height="fill_parent"
    >

    <EditText
        android:layout_width="fill_parent"
        android:layout_height="wrap_content"
        android:id="@+id/todoedit"
        android:layout_weight="0"
    />

    <ScrollView
        android:layout_width="fill_parent"
        android:layout_height="wrap_content"
        android:layout_weight="10"
    >

        <TextView  
            android:layout_width="fill_parent" 
            android:layout_height="wrap_content" 
            android:text="I have nothing"
            android:id="@+id/tododisplay"
        />
    </ScrollView>
    <Button
        android:layout_width="fill_parent"
        android:layout_height="wrap_content"
        android:text="Add item"
        android:id="@+id/btnadd"
        android:onClick="recAdd"
        android:layout_weight="0"
/>
</LinearLayout>
