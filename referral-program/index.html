Referral program (for workshops)

What

Earn PhP 2,000 if you refer someone to take the Android Programming Workshop. Earn PhP 3,000 if you refer someone to take the Java + Android Programming Workshop. These figures are a little more than 10% of the course fee(s).

How

There are two ways;

  1. Send the person my way by simply providing me his name and email address. I will include your name on the email as an introduction. I will take care of the rest

  2. You could save me sometime and point the person directly to either Java + Android Programming Workshop or Android Programming Workshop, then send me the email and name of the person.

The bounty will be paid if the person takes the course. I can hand you over the bounty in a bag of cash or PayPal or a cheque---whichever you prefer.

When

This is happening now. There are scheduled workshops for August and September 2012. The exact dates of the workshop is a little tricky to pin down because it depends on the number of participants.

Who

This program is not open to the general public, that is why I did not put up a link to this page on any part of the website. These are the reasons;

  1. We need to be personally acquainted

  2. You need to be familiar with my skills, because you could be asked about the credibility of the one providing the workshop. You won't be able to say anything if we don't know each other personally

  3. If you don't know me personally, it will be a little difficult for you to trust the offer for the bounty

Some background about the workshops

Android Programming Workshop is 24 hour workshop geared towards people who already know how to code in Java. It is priced at PhP 18,000. The link provided contains a lot of details about the workshop

Java + Android Programming Workshop is a 48 hour workshop which does not require any prior knowledge of Java. This is aimed at beginners. Workshop is priced at PhP 26,000.
